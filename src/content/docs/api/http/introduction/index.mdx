## Overview

SAMA Chat Server provide HTTPS API - a set of administrative-pourpose methods to perform various operations within server.

## Authorization

Each request to HTTPS API must contain `Admin-Api-Key` header which is equal to `HTTP_ADMIN_API_KEY` env.
