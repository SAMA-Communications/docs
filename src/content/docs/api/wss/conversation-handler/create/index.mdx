## Overview

> <Badge text="wss" variant="tip" size="large" />
> <Badge text="conversation_handler_create" size="large" />

The `conversation_handler_create` request registers the schema for the specified chat in the system.

## Request Parameters

| Name      | Type     | Description                  |
| --------- | -------- | ---------------------------- |
| `content` | `string` | Handler logic in string form |
| `cid`     | `string` | Converstaion ID              |

## Response Fields

<Tabs>
  <TabItem label="Response">
```json
{
  "id": "RequestId",
  "success": true
}
```
  </TabItem>
</Tabs>
