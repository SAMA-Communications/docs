## Description

> <Badge text="wss" variant="tip" size="large" />
> <Badge text="conversation_handler_get" size="large" />

The `conversation_handler_get` request returns the chat handle for the specified CID.

## Request Parameters

| Name  | Type     | Description     |
| ----- | -------- | --------------- |
| `cid` | `string` | Converstaion ID |

## Response Fields

| Name         | Type     | Description                  |
| ------------ | -------- | ---------------------------- |
| `updated_by` | `string` | User ID that last modified   |
| `content`    | `string` | Handler logic in string form |
