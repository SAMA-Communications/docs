## Description

> <Badge text="wss" variant="tip" size="large" />
> <Badge text="user_last_activity" size="large" />

This request will help you obtain information about the `last activity` of other users.

## Request Parameters

| Name  | Type            | Description |
| ----- | --------------- | ----------- |
| `ids` | `array[string]` | User IDs    |

## Response Fields

| Name            | Type     | Description               |
| --------------- | -------- | ------------------------- |
| `last_activity` | `object` | Newly created user object |

> Returns the UID and related last activity.

## Example

<Tabs>
  <TabItem label="Request">
```json
{
  "request": {
    "user_last_activity": {
      "ids": ["UID1", "UID2", "UID3"] // max 10 UIDs per request
    },
    "id": "RequestId"
  }
}
```
  </TabItem>  
</Tabs>

<Tabs>
  <TabItem label="Response">
```json
{
  "response": {
    "id": "RequestId",
    "last_activity": {
      "UID1": 1755065166,
      "UID2": 0, // "online"
      "UID3": 1755047354
    }
  }
}
```
  </TabItem>  
</Tabs>
