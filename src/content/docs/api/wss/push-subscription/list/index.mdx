## Description

> <Badge text="wss" variant="tip" size="large" />
> <Badge text="push_subscription_list" size="large" />

The `push_subscription_list` request returns an array of active device subscriptions for a specific user.

## Request Parameters

| Name      | Type     | Description |
| --------- | -------- | ----------- |
| `user_id` | `string` | User ID     |

## Response Fields

| Name            | Type            | Description                             |
| --------------- | --------------- | --------------------------------------- |
| `subscriptions` | `array[object]` | Array of devices for push notifications |
