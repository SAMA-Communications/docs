## Description

> <Badge text="wss" variant="tip" size="large" />
> <Badge text="contact_update" size="large" />

The `contact_update` request allows you to update the specified fields in the contact object.

## Request & Response Fields

All the same as in the `contact_add` request.

<LinkCard title="Address Book / Add" href="/api/wss/address-book/add/" />

## Example

<Tabs>
  <TabItem label="Request">
```json
{
  "request": {
    "contact_update": {
      "id": "63480e68f4794709f802a2fa",
      "first_name": "Name",
      "last_name": "Surname",
      "company": "UserCompany",
      "email": [
        { "type": "work", "value": "..." },
        { "type": "home", "value": "..." },
        ...
      ],
      "phone": [
        { "type": "work", "value": "..." },
        { "type": "home", "value": "..." },

      ]

   }
}
```
