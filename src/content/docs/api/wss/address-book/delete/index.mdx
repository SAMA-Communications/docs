## Description

> <Badge text="wss" variant="tip" size="large" />
> <Badge text="contact_delete" size="large" />

The `contact_delete` request will delete a contact with a specific ID.

## Request Parameters

| Name | Type     | Description |
| ---- | -------- | ----------- |
| `id` | `string` | Contact ID  |

## Response Fields

<Tabs>
  <TabItem label="Response">
```json
{
  "response": {
    "id": "RequestId",
    "success": true
  }
}
```
  </TabItem>  
</Tabs>
