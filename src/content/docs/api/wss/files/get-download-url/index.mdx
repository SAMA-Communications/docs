## Description

> <Badge text="wss" variant="tip" size="large" />
> <Badge text="get_file_urls" size="large" />

The `get_file_urls` request will return an array of links to download the file from the server for rendering on the client.

## Request Parameters

| Name       | Type            | Description       |
| ---------- | --------------- | ----------------- |
| `file_ids` | `array[string]` | Array of file IDs |

## Response Fields

| Name        | Type            | Description        |
| ----------- | --------------- | ------------------ |
| `file_urls` | `array[string]` | Array of file URLs |
