## Overview

> <Badge text="wss" variant="tip" size="large" />
> <Badge text="user_create" size="large" />

The `user_create` request creates a new user.

## Request Parameters

| Name              | Type     | Description                           |
| ----------------- | -------- | ------------------------------------- |
| `organization_id` | `string` | **Identifier** of the organization    |
| `login`           | `string` | Desired **login** name for the user   |
| `password`        | `string` | **Password** for the new user account |
| `email`           | `string` | User's **email** address              |
| `phone`           | `string` | User's **phone** number               |

## Response Fields

| Name   | Type     | Description               |
| ------ | -------- | ------------------------- |
| `user` | `object` | Newly created user object |
