## Description

> <Badge text="wss" variant="tip" size="large" />
> <Badge text="get_users_by_ids" size="large" />

The `get_users_by_ids` request returns user objects by the requested IDs.

## Request Parameters

| Name  | Type            | Description   |
| ----- | --------------- | ------------- |
| `ids` | `array[string]` | Array of UIDs |

## Response Fields

| Name    | Type            | Description           |
| ------- | --------------- | --------------------- |
| `users` | `array[object]` | Array of user objects |

## Example

<Tabs>
  <TabItem label="Request">
```json
{
  "request": {
    "get_users_by_ids": {
      "ids": ["65fcb3f67dbaace5021595b9", "65fcb3f67dbaace502159222"]
    },
    "id": "4"
  }
}
```
  </TabItem>  
</Tabs>

<Tabs>
  <TabItem label="Response">
```json
{
  "response": {
    id: "4",
    "users": [
      { "_id":"65fcb3f67dbaace5021595b9", ... },
      {"_id": "65fcb3f67dbaace502159222", ...}
    ]
  }
}
```
  </TabItem>  
</Tabs>
