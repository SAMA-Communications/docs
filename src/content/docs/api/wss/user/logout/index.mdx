## Description

> <Badge text="POST" variant="caution" size="large" />
> <Badge text="/logout" size="large" />

The request ends the user's active session on the current device.

## Request & Response Fields

The `body` of the request can be empty, but you need to put the `access_token` in the request header, for example:

<Tabs>
  <TabItem label="Request & Header">
    ```json 
    { Authorization: `Bearer ${accessToken}` }
    ```
  </TabItem>
</Tabs>

<Tabs>
  <TabItem label="Response">
```json
{ "success": true }
```
  </TabItem>  
</Tabs>

<Aside>
  If you do not pass an actual `refresh_token` in the http request credentials,
  then use hard session cleanup using a web socket request.
</Aside>
