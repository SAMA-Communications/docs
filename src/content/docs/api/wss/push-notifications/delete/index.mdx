## Overview

> <Badge text="wss" variant="tip" size="large" />
> <Badge text="push_subscription_delete" size="large" />

The `push_subscription_delete` request deletes the active subscription of the device whose ID was passed in the body of the request.

## Request Parameters

| Name                                   | Type     | Description            |
| -------------------------------------- | -------- | ---------------------- |
| `push_subscription_delete.device_udid` | `string` | Subscription device id |

## Response Fields

<Tabs>
  <TabItem label="Response">
```json
{

  "success": true
}
```
  </TabItem>
</Tabs>
