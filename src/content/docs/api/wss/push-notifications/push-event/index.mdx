## Overview

> <Badge text="wss" variant="tip" size="large" />
> <Badge text="push_event_create" size="large" />

The `push_event_create` request —Åreates a custom event to send.

## Request Parameters

| Name             | Type            | Description            |
| ---------------- | --------------- | ---------------------- |
| `recipients_ids` | `array[string]` | Array of recipient IDs |
| `message`        | `object`        | Message object         |

## Response Fields

| Name                   | Type     | Description              |
| ---------------------- | -------- | ------------------------ |
| `event.message`        | `string` | Message in string format |
| `event.recipients_ids` | `string` | Array of recipients IDs  |
