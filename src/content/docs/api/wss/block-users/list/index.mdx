## Description

> <Badge text="wss" variant="tip" size="large" />
> <Badge text="list_blocked_users" size="large" />

The `list_blocked_users` request will return a list of blocked users for the current session user.

## Request Parameters

| Name | Type | Description |
| ---- | ---- | ----------- |
| -    | -    | -           |

## Response Fields

| Name    | Type      | Description              |
| ------- | --------- | ------------------------ |
| `users` | `boolean` | List of blocked user IDs |

## Example

<Tabs>
  <TabItem label="Request">
    ```json
    {
      "request": {
        "list_blocked_users": {},
        "id": "RequestId"
      }
    }
    ```
  </TabItem>  
</Tabs>

<Tabs>
  <TabItem label="Response">
    ```json
    {
      "response": {
        "id": "RequestId",
        "users": ["63077ad836b78c3d82af0812", "63077ad836b78c3d82af0813"]
      }
    }
    ```
  </TabItem>  
</Tabs>
