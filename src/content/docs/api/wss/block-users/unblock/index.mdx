## Overview

> <Badge text="wss" variant="tip" size="large" />
> <Badge text="unblock_user" size="large" />

The `unblock_user` request will unblock the sending of packets to users who have identifiers from the sent array.

## Request Parameters

| Name  | Type            | Description |
| ----- | --------------- | ----------- |
| `ids` | `array[string]` | User IDs    |

## Response Fields

<Tabs>
  <TabItem label="Response">
```json
{
  "response": {
    "id": "RequestId",
    "success": true
  }
}
```
  </TabItem>  
</Tabs>
