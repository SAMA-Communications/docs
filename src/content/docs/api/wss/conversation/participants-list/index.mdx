## Description

> <Badge text="wss" variant="tip" size="large" />
> <Badge text="get_participants_by_cids" size="large" />

The `get_participants_by_cids` request returns a list of users upon request. The conversations field indicates which user is from which chat, which is useful if the user is part of several different chats.

## Request Parameters

| Name   | Type            | Description               |
| ------ | --------------- | ------------------------- |
| `cids` | `array[string]` | Array of Conversation IDs |

## Response Fields

| Name            | Type            | Description                                             |
| --------------- | --------------- | ------------------------------------------------------- |
| `users`         | `array[object]` | Array of User objects                                   |
| `conversations` | `object`        | Object of Conversation IDS with users belonging to them |
