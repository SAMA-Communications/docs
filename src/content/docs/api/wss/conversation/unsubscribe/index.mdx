## Overview

> <Badge text="wss" variant="tip" size="large" />
> <Badge text="conversation_unsubscribe" size="large" />

The `conversation_unsubscribe` request allowed the user to unsubscribe from the channel (leave).

## Request Parameters

| Name  | Type     | Description                      |
| ----- | -------- | -------------------------------- |
| `cid` | `string` | Conversation ID fot subscription |
