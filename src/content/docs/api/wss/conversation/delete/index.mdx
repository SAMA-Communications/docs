## Overview

> <Badge text="wss" variant="tip" size="large" />
> <Badge text="conversation_delete" size="large" />

The `conversation_delete` request willl delete Convarsation by specified ID.

## Request Parameters

| Name | Type     | Description                  |
| ---- | -------- | ---------------------------- |
| `id` | `string` | Conversation ID for deletion |

## Response Fields

<Tabs>
  <TabItem label="Response">
```json
{
  "id": "RequestId",
  "success": true
}
```
  </TabItem>
</Tabs>

## Example

<Tabs>
  <TabItem label="Request">
```json
{
  "request": {
    "conversation_delete": {
      "id": "63077ad836b78c3d82af0812"
    },
    "id": "RequestId"
  }
}
```
  </TabItem>  
</Tabs>
