## Description

> <Badge text="wss" variant="tip" size="large" />
> <Badge text="conversation_subscribe" size="large" />

The `conversation_subscribe` request allowed the user to subscribe to the channel (receive/read messages).

## Request Parameters

| Name  | Type     | Description                      |
| ----- | -------- | -------------------------------- |
| `cid` | `string` | Conversation ID fot subscription |
