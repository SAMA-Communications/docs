## Description

> <Badge text="wss" variant="tip" size="large" />
> <Badge text="conversation_search" size="large" />

The `conversation_search` request returns a list of chats based on the specified parameters.

## Request Parameters

| Name            | Type        | Description           |
| --------------- | ----------- | --------------------- |
| `name`          | `string`    | Key phrase for search |
| `limit`         | `number`    | Max number of objects |
| `updated_at.gt` | `timestamp` | Time reference point  |

## Response Fields

| Name            | Type            | Description                   |
| --------------- | --------------- | ----------------------------- |
| `conversations` | `array[object]` | Array of Conversation objects |

## Example

<Tabs>
  <TabItem label="Request">
```json
{
  "request": {
    "conversation_search": {
      "name": "conversationName",
      "limit": 100,
      "updated_at": {
        "gt": timestamp_in_ms,
      },
    },
    "id": "RequestId"
  }
}
```
  </TabItem>  
</Tabs>
