## Overview

> <Badge text="wss" variant="tip" size="large" />
> <Badge text="message_reactions_update" size="large" />

The `message_reactions_update` request allowed users who had access to the message to add/delete reactions.

## Request Parameters

| Name     | Type     | Description                    |
| -------- | -------- | ------------------------------ |
| `mid`    | `string` | Identifier of the message      |
| `add`    | `string` | Emoji that needs to be saved   |
| `remove` | `string` | Emoji that needs to be removed |

## Response Fields

<Tabs>
  <TabItem label="Response">
```json
{
  "id": "RequestId",
  "success": true
}
```
  </TabItem>
</Tabs>

## Example

<Tabs>
  <TabItem label="Request">
```json
{
  "request": {
    "message_reactions_update": {
      "mid": "63077ad836b78c3d82af0812",
      "add": "👍",
      "remove": "😿"
    },
    "id": "RequestId"
  }
}
```
  </TabItem>  
</Tabs>
